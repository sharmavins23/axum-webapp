# Rust: Axum WebApp

This project is a follow-along of
[Jeremy Chone's Rust Production Coding: Web App](https://www.youtube.com/playlist?list=PL7r-PXl6ZPcCTTxjmsb9bFZB9i01fAtI7)

different features.
